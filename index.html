搭建个人博客记录

================

  1. 打开网址nodejs.org下载如下图所示安装包，安装过程中一直点next即可

    1. https://git-scm.com/downloads 打开网址安装git ，一切都是默认设置，只管点next

3.安装完毕之后，在你想放的位置，创建一个文件夹，里面将要存放以后搭建博客的一些文件，之后的操作都将在这个文件夹内。之后如下图所示操作 会出现一个命令窗口

4.在命令窗口输入 npm install -g cnpm –registry=https /registry.npm.taobao.org 回车 ，下载cnpm，这是为了之后下载能更快,之后再输入 hexo init，下载安装搭建博客需要的一些初始文件

5.要把自己的个人博客部署到远端服务器，使得别人能够通过远端服务器访问自己的博客网页，否则只有自己能在本机看到博客，为此需要安装一个部署插件 在命令窗口输入

cnpm install –save hexo-deployer-git

6.接下来的步骤就是对SSH的配置，这里是将个人博客搭建在github上，首先你得需要创建一个github账号，注意创建账号时建议用手机的最新版本谷歌浏览器，因为这个网站对一些浏览器不支持，电脑上创建有的也会出现问题

<1>.如下图，点击

<2>.

![](C:\Users\86187\Desktop\博客图片\6 (2).png)

<3>.再回到 git的命令编辑界面 输入 vim _config.yml ，之后按i或者a 进入文本编辑模式，拉到最底下，将内容设置成如下图所示，编辑完成后按esc,再按住shift+冒号键，在最下端输入wq保存退出，该步骤你也可以直接打开 _config.yml文件对它进行编辑，

<4>.将本地端的个人博客 上传至远端

依次输入命令 hexo clean （清除缓存） , hexo g（生成文件），hexo d(上传至远端服务器)；

PS：在这个过程中 可能会出现 please tell me who are you 的报错，此时你是不能讲文档上传至远端服务器的

你按照它的提示在命令窗口输入

*git config user.name “username”———————-username就是你的github用户名

*git config user.email “email” ———————————email就是你创建账号时输入的邮箱，”前面是有空格的

结束语：

之后还有如何设置主题等一些细节上的操作可自行百度，另外怎么写博客呢，不像我们用wps文档记录可以直接插入图片编辑文字，我们要需要一个文档编辑器，按照makedown的格式进行书写。

编辑器我用typore，https://www.cnblogs.com/-guz/p/10258557.html 这篇文章里有提供下载地址，另外还有一些快捷操作，此外https://www.runoob.com/markdown/md-tutorial.html这个网页提供了makedown格式的一些书写语法.

可能至此可能你还不知道makedown是啥，按照我的理解其实他就是一种写作格式吧，在支持makedown格式的文档编辑器按照makedown的格式编辑 我们想记录的，之后他就可以上传到远端，大家都可以看到了。
